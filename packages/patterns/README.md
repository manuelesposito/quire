# @quire/patterns

Composed blocks built from `@quire/components`: the recurring page-level shapes of the admin
— **SettingsPage**, **ListTablePage**, **Dashboard**, and the **metabox→card unifier** that
collapses classic postboxes and modern cards into one container.

Patterns are where the four-languages-into-one convergence becomes visible at the page level.
