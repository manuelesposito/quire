# @quire/primitives

Unstyled, accessible **behaviors** — the mechanics of a button, toggle, dialog, tabs, table,
and menu with full keyboard and ARIA support, and **no visual opinion**. `@quire/components`
adds the Quire look on top of these.

Keeping behavior and look separate means accessibility is solved once, centrally, and every
styled component inherits it.
