# @quire/components

The styled Quire library: `@quire/primitives` (behavior + a11y) dressed in `@quire/tokens`
(the look). This is where "one button," "one settings row," "one notice" actually live.

## First components (the decisive proof)

The first deliverable is small on purpose. These three, rendered against three
deliberately-conflicting real screens, decide whether the system is real:

- **Button** — primary / secondary / tertiary / destructive.
- **SettingsRow** — label + control + help text + help-tip.
- **Notice** — success / info / warning / error.

If one Button and one SettingsRow can make WP General Settings, WooCommerce Settings, and
Jetpack Security look like one product, Quire works.
