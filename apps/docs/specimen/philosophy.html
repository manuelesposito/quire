The design language

What Quire is for

An everyday person's WordPress admin that feels like a well-made book — calm, coherent, and cared-for — not a committee's dashboard.
The name is the idea

A quire is a bookbinding term: a gathering of folded sheets, bound into a book.

WordPress's admin today is loose, mismatched pages from four different printers — wp-admin, Gutenberg, WooCommerce, Jetpack — never bound together. Quire's whole reason for being is to gather the scattered pages into one coherent, well-bound whole.

What we believe
  • Coherence is respect.
    Every clashing button is a small tax on someone's attention and confidence. Making everything mean one thing is how we respect the person using it.
  • Calm is the feature.
    The work is subtraction — fewer colours, fewer styles, less noise. A quiet screen is a finished screen. We never add a control to solve a problem we could remove.
  • Made like a book, not a form.
    Warm paper, a real typeface, craft in the small things. Software can carry the dignity of a well-made physical object — and it should, because the people using it deserve that care.
  • For the everyday owner.
    Not developers, not agencies — the shop owner, the writer, the volunteer who never chose to learn an admin panel. They deserve one that doesn't punish them.
  • Built to age, not to trend.
    Bookish over fashionable — meant to feel right in ten years, not novel for one. This is why "not generic" matters: generic is disposable.
  • A commons, not a product.
    Open, GPL, owned by the community that runs a large share of the web. Quire answers to no vendor.
The feel

Warm putty paper, not white — the colour of an old page. Surfaces separate by tone and a hairline, never by drop shadow; shadow is saved for things that truly float. A muted, vintage almanac palette where one warm accent does the emphasis and nothing else shouts. Generous spacing, few things on screen at once. The emotion we're aiming for is reassurance — the calm of a well-made book, for someone who never asked to learn an admin panel.

Typography — three voices, three jobs

Type carries most of the feeling. Quire uses three families, each with one clear job, and keeps strict discipline between them — that restraint is what keeps a dense screen calm.

Newsreader — the voice
What a cleaner WordPress admin feels like
warm, calm, and deliberate
A warm, literary serif made for reading on screens, with optical sizing so it stays graceful from a caption to a headline. It carries Quire's character: page titles, display headings, and the italic section labels that read like a book's running-heads. Reserved on purpose — used where character matters, so it never becomes wallpaper.

Inter — the work
The everyday working text — calm and dense, perfectly legible at small sizes.
Buttons, labels, table data, settings rows, help text — the heavy lifting.
A neutral, humanist sans built for interfaces: exceptionally legible small, calm in dense layouts. It does the everyday work so the serif can stay special. If the serif is Quire's voice, Inter is its hands.
Inter · sans-serif · Google Fonts · weights 400 / 450 / 500 / 600 · the UI workhorse

A serif for soul, a humanist sans for the work, a mono for the machinery.

The full size scale, weights, and named text styles (display, page-title, section, body, label, caption, mono-label, code) live on the Tokens screen, where you can see each one resolved live.

Where we're going

Today Quire is a coherent skin over the existing admin. The North Star is a genuine reimagining of what running a WordPress site feels like. At every stage the immutable core is the feeling — calm, bound, warm, made with care.

The detailed rules and the locked grammar live in DESIGN.md. This page is its front door — the why, before the what.
